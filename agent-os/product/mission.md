# Mission

## Problem

Australians lack a self-hosted, privacy-respecting way to track personal finances using real bank data. Existing solutions are either cloud-hosted (raising data ownership concerns) or require manual transaction entry.

## Target Users

Australian individuals who want to budget using real bank transaction data while maintaining full control over their financial data (self-hosted).

## Solution

A self-hosted personal budgeting app that pulls real bank data via Basiq's CDR-accredited open banking API, auto-categorizes transactions using ANZSIC codes, and renders interactive financial dashboards — all without surrendering data to a third-party SaaS.
